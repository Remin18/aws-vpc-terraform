### ディレクトリ構造
```
.
├── main.tf
├── modules
│   └── vpc
│       ├── main.tf
│       └── variables.tf
├── terraform.tfvars
└── variables.tf
```

### ローカルのワークスペースを初期化
```
terraform init
```

### 変更リソースの確認
```
terraform plan
```

### 変更リソースの確認
```
terraform apply
```

### リソースの破棄
```
terraform destroy
```
